    ）
   </rp>
  </ruby>
  だという評判のあるのが、鼻の先で、せせら笑いながら、
  <br/>
  「君はこの金貨を元の石炭にしようと言う。僕たちはまたしたくないと言う。それじゃいつまでたった所で、議論が
  <ruby>
   <rb>
    干
   </rb>
   <rp>
    （
   </rp>
   <rt>
    ひ
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  ないのは当り前だろう。そこで僕が思うには、この金貨を元手にして、君が僕たちと
  <ruby>
   <rb>
    骨牌
   </rb>
   <rp>
    （
   </rp>
   <rt>
    かるた
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  をするのだ。そうしてもし君が勝ったなら、石炭にするとも何にするとも、自由に君が始末するが
  <ruby>
   <rb>
    好
   </rb>
   <rp>
    （
   </rp>
   <rt>
    い
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  い。が、もし僕たちが勝ったなら、金貨のまま僕たちへ渡し給え。そうすれば御互の申し分も立って、至極満足だろうじゃないか。」
  <br/>
  それでも私はまだ首を振って、容易にその申し出しに賛成しようとはしませんでした。所がその友人は、いよいよ
  <ruby>
   <rb>
    嘲
   </rb>
   <rp>
    （
   </rp>
   <rt>
    あざけ
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  るような
  <ruby>
   <rb>
    笑
   </rb>
   <rp>
    （
   </rp>
   <rt>
    えみ
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  を浮べながら、私とテエブルの上の金貨とを
  <ruby>
   <rb>
    狡
   </rb>
   <rp>
    （
   </rp>
   <rt>
    ず
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  るそうにじろじろ見比べて、
  <br/>
  「君が僕たちと
  <ruby>
   <rb>
    骨牌
   </rb>
   <rp>
    （
   </rp>
   <rt>
    かるた
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  をしないのは、つまりその金貨を僕たちに取られたくないと思うからだろう。それなら魔術を使うために、欲心を捨てたとか何とかいう、
  <ruby>
   <rb>
    折角
   </rb>
   <rp>
    （
   </rp>
   <rt>
    せっかく
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  の君の決心も怪しくなってくる訳じゃないか。」
  <br/>
  「いや、何も僕は、この金貨が惜しいから石炭にするのじゃない。」
  <br/>
  「それなら
  <ruby>
   <rb>
    骨牌
   </rb>
   <rp>
    （
   </rp>
   <rt>
    かるた
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  をやり給えな。」
  <br/>
  何度もこういう押問答を繰返した後で、とうとう私はその友人の言葉通り、テエブルの上の金貨を
  <ruby>
   <rb>
    元手
   </rb>
   <rp>
    （
   </rp>
   <rt>
    もとで
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  に、どうしても
  <ruby>
   <rb>
    骨牌
   </rb>
   <rp>
    （
   </rp>
   <rt>
    かるた
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  を闘わせなければならない
  <ruby>
   <rb>
    羽目
   </rb>
   <rp>
    （
   </rp>
   <rt>
    はめ
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  に立ち至りました。勿論友人たちは皆大喜びで、すぐにトランプを一組取り寄せると、部屋の片隅にある
  <ruby>
   <rb>
    骨牌机
   </rb>
   <rp>
    （
   </rp>
   <rt>
    かるたづくえ
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  を囲みながら、まだためらい勝ちな私を早く早くと
  <ruby>
   <rb>
    急
   </rb>
   <rp>
    （
   </rp>
   <rt>
    せ
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  き立てるのです。
  <br/>
  ですから私も仕方がなく、しばらくの間は友人たちを相手に、
  <ruby>
   <rb>
    嫌々
   </rb>
   <rp>
    （
   </rp>
   <rt>
    いやいや
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  <ruby>
   <rb>
    骨牌
   </rb>
   <rp>
    （
   </rp>
   <rt>
    かるた
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  をしていました。が、どういうものか、その夜に限って、ふだんは格別
  <ruby>
   <rb>
    骨牌
   </rb>
   <rp>
    （
   </rp>
   <rt>
    かるた
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  上手でもない私が、嘘のようにどんどん勝つのです。するとまた妙なもので、始は気のりもしなかったのが、だんだん面白くなり始めて、ものの十分とたたない内に、いつか私は一切を忘れて、熱心に
  <ruby>
   <rb>
    骨牌
   </rb>
   <rp>
    （
   </rp>
   <rt>
    かるた
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  を引き始めました。
  <br/>
  友人たちは、元より私から、あの金貨を残らず
  <ruby>
   <rb>
    捲
   </rb>
   <rp>
    （
   </rp>
   <rt>
    ま
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  き上げるつもりで、わざわざ
  <ruby>
   <rb>
    骨牌
   </rb>
   <rp>
    （
   </rp>
   <rt>
    かるた
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  を始めたのですから、こうなると皆あせりにあせって、ほとんど
  <ruby>
   <rb>
    血相
   </rb>
   <rp>
    （
   </rp>
   <rt>
    けっそう
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  さえ変るかと思うほど、夢中になって勝負を争い出しました。が、いくら友人たちが躍起となっても、私は一度も負けないばかりか、とうとうしまいには、あの金貨とほぼ同じほどの
  <ruby>
   <rb>
    金高
   </rb>
   <rp>
    （
   </rp>
   <rt>
    きんだか
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  だけ、私の方が勝ってしまったじゃありませんか。するとさっきの人の悪い友人が、まるで、気違いのような勢いで、私の前に、
  <ruby>
   <rb>
    札
   </rb>
   <rp>
    （
   </rp>
   <rt>
    ふだ
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  をつきつけながら、
  <br/>
  「さあ、引き給え。僕は僕の財産をすっかり賭ける。地面も、
  <ruby>
   <rb>
    家作
   </rb>
   <rp>
    （
   </rp>
   <rt>
    かさく
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  も、馬も、自働車も、一つ残らず賭けてしまう。その代り君はあの金貨のほかに、今まで君が勝った金をことごとく賭けるのだ。さあ、引き給え。」
  <br/>
  私はこの
  <ruby>
   <rb>
    刹那
   </rb>
   <rp>
    （
   </rp>
   <rt>
    せつな
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  に欲が出ました。テエブルの上に積んである、山のような金貨ばかりか、折角私が勝った金さえ、今度運悪く負けたが最後、皆相手の友人に取られてしまわなければなりません。のみならずこの勝負に勝ちさえすれば、私は向うの全財産を一度に手へ入れることが出来るのです。こんな時に使わなければどこに魔術などを教わった、苦心の
  <ruby>
   <rb>
    甲斐
   </rb>
   <rp>
    （
   </rp>
   <rt>
    かい
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  があるのでしょう。そう思うと私は
  <ruby>
   <rb>
    矢
   </rb>
   <rp>
    （
   </rp>
   <rt>
    や
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  も
  <ruby>
   <rb>
    楯
   </rb>
   <rp>
    （
   </rp>
   <rt>
    たて
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  もたまらなくなって、そっと魔術を使いながら、決闘でもするような勢いで、
  <br/>
  「よろしい。まず君から引き給え。」
  <br/>
  「
  <ruby>
   <rb>
    九
   </rb>
   <rp>
    （
   </rp>
   <rt>
    く
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  。」
  <br/>
  「
  <ruby>
   <rb>
    王様
   </rb>
   <rp>
    （
   </rp>
   <rt>
    キング
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  。」
  <br/>
  私は勝ち誇った声を挙げながら、まっ蒼になった相手の眼の前へ、引き当てた
  <ruby>
   <rb>
    札
   </rb>
   <rp>
    （
   </rp>
   <rt>
    ふだ
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  を出して見せました。すると不思議にもその
  <ruby>
   <rb>
    骨牌
   </rb>
   <rp>
    （
   </rp>
   <rt>
    かるた
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  の
  <ruby>
   <rb>
    王様
   </rb>
   <rp>
    （
   </rp>
   <rt>
    キング
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  が、まるで魂がはいったように、
  <ruby>
   <rb>
    冠
   </rb>
   <rp>
    （
   </rp>
   <rt>
    かんむり
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  をかぶった頭を
  <ruby>
   <rb>
    擡
   </rb>
   <rp>
    （
   </rp>
   <rt>
    もた
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  げて、ひょいと
  <ruby>
   <rb>
    札
   </rb>
   <rp>
    （
   </rp>
   <rt>
    ふだ
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  の外へ体を出すと、行儀よく剣を持ったまま、にやりと気味の悪い微笑を浮べて、
  <br/>
  「御婆サン。御婆サン。御客様ハ御帰リニナルソウダカラ、寝床ノ仕度ハシナクテモ好イヨ。」
  <br/>
  と、聞き覚えのある声で言うのです。と思うと、どういう訳か、窓の外に降る
  <ruby>
   <rb>
    雨脚
   </rb>
   <rp>
    （
   </rp>
   <rt>
    あまあし
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  までが、急にまたあの大森の竹藪にしぶくような、寂しいざんざ
  <ruby>
   <rb>
    降
   </rb>
   <rp>
    （
   </rp>
   <rt>
    ぶ
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  りの音を立て始めました。
  <br/>
  ふと気がついてあたりを見廻すと、私はまだうす暗い石油ランプの光を浴びながら、まるであの
  <ruby>
   <rb>
    骨牌
   </rb>
   <rp>
    （
   </rp>
   <rt>
    かるた
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  の
  <ruby>
   <rb>
    王様
   </rb>
   <rp>
    （
   </rp>
   <rt>
    キング
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  のような微笑を浮べているミスラ君と、向い合って坐っていたのです。
  <br/>
  私が指の間に
  <ruby>
   <rb>
    挟
   </rb>
   <rp>
    （
   </rp>
   <rt>
    はさ
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  んだ葉巻の灰さえ、やはり落ちずにたまっている所を見ても、私が一月ばかりたったと思ったのは、ほんの二三分の間に見た、夢だったのに違いありません。けれどもその二三分の短い間に、私がハッサン・カンの魔術の秘法を習う資格のない人間だということは、私自身にもミスラ君にも、明かになってしまったのです。私は恥しそうに頭を下げたまま、しばらくは口もきけませんでした。
  <br/>
  「私の魔術を使おうと思ったら、まず欲を捨てなければなりません。あなたはそれだけの修業が出来ていないのです。」
  <br/>
  ミスラ君は気の毒そうな眼つきをしながら、縁へ赤く花模様を織り出したテエブル掛の上に
  <ruby>
   <rb>
    肘
   </rb>
   <rp>
    （
   </rp>
   <rt>
    ひじ
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  をついて、静にこう私をたしなめました。
