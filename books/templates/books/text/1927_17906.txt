  「この香水はへんに酢くさい。どうしたんだらう。」
  <br/>
  「まちがへたんだ。下女が
  <ruby>
   <rb>
    風邪
   </rb>
   <rp>
    （
   </rp>
   <rt>
    かぜ
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  でも引いてまちがへて入れたんだ。」
  <br/>
  二人は扉をあけて中にはひりました。
  <br/>
  扉の裏側には、大きな字で斯う書いてありました。
  <br/>
  <div class="jisage_3" style="margin-left: 3em">
   「いろいろ注文が多くてうるさかつたでせう。お気の毒でした。
   <br/>
   もうこれだけです。どうかからだ中に、
   <ruby>
    <rb>
     壺
    </rb>
    <rp>
     （
    </rp>
    <rt>
     つぼ
    </rt>
    <rp>
     ）
    </rp>
   </ruby>
   の中の塩をたくさ
   <br/>
   んよくもみ込んでください。」
   <br/>
  </div>
  なるほど立派な青い瀬戸の塩壺は置いてありましたが、こんどといふこんどは二人ともぎよつとしてお互にクリームをたくさん塗つた顔を見合せました。
  <br/>
  「どうもをかしいぜ。」
  <br/>
  「ぼくもをかしいとおもふ。」
  <br/>
  「沢山の注文といふのは、向ふがこつちへ注文してるんだよ。」
  <br/>
  「だからさ、西洋料理店といふのは、ぼくの考へるところでは、西洋料理を、来た人にたべさせるのではなくて、来た人を西洋料理にして、食べてやる
  <ruby>
   <rb>
    家
   </rb>
   <rp>
    （
   </rp>
   <rt>
    うち
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  とかういふことなんだ。これは、その、つ、つ、つ、つまり、ぼ、ぼ、ぼくらが……。」がたがたがたがた、ふるへだしてもうものが言へませんでした。
  <br/>
  「その、ぼ、ぼくらが、……うわあ。」がたがたがたがたふるへだして、もうものが言へませんでした。
  <br/>
  「
  <ruby>
   <rb>
    遁
   </rb>
   <rp>
    （
   </rp>
   <rt>
    に
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  げ……。」がたがたしながら一人の紳士はうしろの戸を押さうとしましたが、どうです、戸はもう
  <ruby>
   <rb>
    一分
   </rb>
   <rp>
    （
   </rp>
   <rt>
    いちぶ
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  も動きませんでした。
  <br/>
  奥の方にはまだ一枚扉があつて、大きなかぎ穴が二つつき、銀いろのホークとナイフの形が切りだしてあつて、
  <br/>
  <div class="jisage_3" style="margin-left: 3em">
   「いや、わざわざご苦労です。
   <br/>
   大へん結構にできました。
   <br/>
   さあさあおなかにおはひりください。」
   <br/>
  </div>
  と書いてありました。おまけにかぎ穴からはきよろきよろ二つの青い
  <ruby>
   <rb>
    眼玉
   </rb>
   <rp>
    （
   </rp>
   <rt>
    めだま
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  がこつちをのぞいてゐます。
  <br/>
  「うわあ。」がたがたがたがた。
  <br/>
  「うわあ。」がたがたがたがた。
  <br/>
  ふたりは泣き出しました。
  <br/>
  すると戸の中では、こそこそこんなことを云つてゐます。
  <br/>
  「だめだよ。もう気がついたよ。塩をもみこまないやうだよ。」
  <br/>
  「あたりまへさ。親分の書きやうがまづいんだ。あすこへ、いろいろ注文が多くてうるさかつたでせう、お気の毒でしたなんて、間抜けたことを書いたもんだ。」
  <br/>
  「どつちでもいゝよ。どうせぼくらには、骨も分けて
  <ruby>
   <rb>
    呉
   </rb>
   <rp>
    （
   </rp>
   <rt>
    く
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  れやしないんだ。」
  <br/>
  「それはさうだ。けれどももしこゝへあいつらがはひつて来なかつたら、それはぼくらの責任だぜ。」
  <br/>
  「呼ばうか、呼ばう。おい、お客さん方、早くいらつしやい。いらつしやい。いらつしやい。お
  <ruby>
   <rb>
    皿
   </rb>
   <rp>
    （
   </rp>
   <rt>
    さら
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  も洗つてありますし、菜つ葉ももうよく塩でもんで置きました。あとはあなたがたと、菜つ葉をうまくとりあはせて、まつ白なお皿にのせる
  <ruby>
   <rb>
    丈
   </rb>
   <rp>
    （
   </rp>
   <rt>
    だ
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  けです。はやくいらつしやい。」
  <br/>
  「へい、いらつしやい、いらつしやい。それともサラドはお嫌ひですか。そんならこれから火を起してフライにしてあげませうか。とにかくはやくいらつしやい。」
  <br/>
  二人はあんまり心を痛めたために、顔がまるでくしやくしやの
  <ruby>
   <rb>
    紙屑
   </rb>
   <rp>
    （
   </rp>
   <rt>
    かみくづ
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  のやうになり、お互にその顔を見合せ、ぶるぶるふるへ、声もなく泣きました。
  <br/>
  中ではふつふつとわらつてまた叫んでゐます。
  <br/>
  「いらつしやい、いらつしやい。そんなに泣いては折角のクリームが流れるぢやありませんか。へい、たゞいま。ぢきもつてまゐります。さあ、早くいらつしやい。」
  <br/>
  「早くいらつしやい。親方がもうナフキンをかけて、ナイフをもつて、舌なめずりして、お客さま方を待つてゐられます。」
  <br/>
  二人は泣いて泣いて泣いて泣いて泣きました。
  <br/>
  そのときうしろからいきなり、
  <br/>
  「わん、わん、ぐわあ。」といふ声がして、あの
  <ruby>
   <rb>
    白熊
   </rb>
   <rp>
    （
   </rp>
   <rt>
    しろくま
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  のやうな犬が二
  <ruby>
   <rb>
    疋
   </rb>
   <rp>
    （
   </rp>
   <rt>
    ひき
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  、
  <ruby>
   <rb>
    扉
   </rb>
   <rp>
    （
   </rp>
   <rt>
    と
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  をつきやぶつて
  <ruby>
   <rb>
    室
   </rb>
   <rp>
    （
   </rp>
   <rt>
    へや
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  の中に飛び込んできました。
  <ruby>
   <rb>
    鍵穴
   </rb>
   <rp>
    （
   </rp>
   <rt>
    かぎあな
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  の眼玉はたちまちなくなり、犬どもはううとうなつてしばらく室の中をくるくる廻つてゐましたが、また一声
  <br/>
  「わん。」と高く
  <ruby>
   <rb>
    吠
   </rb>
   <rp>
    （
   </rp>
   <rt>
    ほ
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  えて、いきなり次の扉に飛びつきました。戸はがたりとひらき、犬どもは吸ひ込まれるやうに飛んで行きました。
  <br/>
  その扉の向ふのまつくらやみのなかで、
  <br/>
  「にやあお、くわあ、ごろごろ。」といふ声がして、それからがさがさ鳴りました。
  <br/>
  室はけむりのやうに消え、二人は寒さにぶるぶるふるへて、草の中に立つてゐました。
  <br/>
  見ると、上着や靴や財布やネクタイピンは、あつちの枝にぶらさがつたり、こつちの根もとにちらばつたりしてゐます。風がどうと吹いてきて、草はざわざわ、木の葉はかさかさ、木はごとんごとんと鳴りました。
  <br/>
  犬がふうとうなつて戻つてきました。
  <br/>
  そしてうしろからは、
  <br/>
  「
  <ruby>
   <rb>
    旦那
   </rb>
   <rp>
    （
   </rp>
   <rt>
    だんな
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  あ、旦那あ、」と叫ぶものがあります。
  <br/>
  二人は
  <ruby>
   <rb>
    俄
   </rb>
   <rp>
    （
   </rp>
   <rt>
    には
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  かに元気がついて
  <br/>
  「おゝい、おゝい、こゝだぞ、早く来い。」と叫びました。
  <br/>
  <ruby>
   <rb>
    簔帽子
   </rb>
   <rp>
    （
   </rp>
   <rt>
    みのばうし
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  をかぶつた専門の猟師が、草をざわざわ分けてやつてきました。

  <br/>
