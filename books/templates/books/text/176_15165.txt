  もかぶらず、曳かれて参ります。どうも物盗りを捕えて、これからその
  <ruby>
   <rb>
    住家
   </rb>
   <rp>
    （
   </rp>
   <rt>
    すみか
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  へ、
  <ruby>
   <rb>
    実録
   </rb>
   <rp>
    （
   </rp>
   <rt>
    じつろく
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  をしに行く所らしいのでございますな。
  <br/>
  「しかも、その物盗りと云うのが、
  <ruby>
   <rb>
    昨夜
   </rb>
   <rp>
    （
   </rp>
   <rt>
    ゆうべ
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  、五条の坂で云いよった、あの男だそうじゃございませぬか。娘はそれを見ると、何故か、涙がこみ上げて来たそうでございます。これは、当人が、手前に話しました――何も、その男に
  <ruby>
   <rb>
    惚
   </rb>
   <rp>
    （
   </rp>
   <rt>
    ほ
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  れていたの、どうしたのと云う訳じゃない。が、その
  <ruby>
   <rb>
    縄目
   </rb>
   <rp>
    （
   </rp>
   <rt>
    なわめ
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  をうけた姿を見たら、急に自分で、自分がいじらしくなって、思わず泣いてしまったと、まあこう云うのでございますがな。まことにその話を聞いた時には、手前もつくづくそう思いましたよ――」
  <br/>
  「何とね。」
  <br/>
  「観音様へ
  <ruby>
   <rb>
    願
   </rb>
   <rp>
    （
   </rp>
   <rt>
    がん
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  をかけるのも考え物だとな。」
  <br/>
  「だが、お
  <ruby>
   <rb>
    爺
   </rb>
   <rp>
    （
   </rp>
   <rt>
    じい
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  さん。その女は、それから、どうにかやって行けるようになったのだろう。」
  <br/>
  「どうにか所か、今では何不自由ない身の上になって居ります。その綾や絹を売ったのを
  <ruby>
   <rb>
    本
   </rb>
   <rp>
    （
   </rp>
   <rt>
    もと
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  に致しましてな。観音様も、これだけは、御約束をおちがえになりません。」
  <br/>
  「それなら、そのくらいな目に遇っても、結構じゃないか。」
  <br/>
  外の日の光は、いつの間にか、黄いろく夕づいた。その中を、風だった竹籔の音が、かすかながらそこここから聞えて来る。往来の人通りも、暫くはとだえたらしい。
  <br/>
  「人を殺したって、物盗りの女房になったって、する気でしたんでなければ仕方がないやね。」
  <br/>
  青侍は、扇を帯へさしながら、立上った。
  <ruby>
   <rb>
    翁
   </rb>
   <rp>
    （
   </rp>
   <rt>
    おきな
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  も、もう
  <ruby>
   <rb>
    提
   </rb>
   <rp>
    （
   </rp>
   <rt>
    ひさげ
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  の水で、泥にまみれた手を洗っている――二人とも、どうやら、暮れてゆく春の日と、相手の心もちとに、物足りない何ものかを、感じてでもいるような
  <ruby>
   <rb>
    容子
   </rb>
   <rp>
    （
   </rp>
   <rt>
    ようす
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  である。
  <br/>
  「とにかく、その女は仕合せ者だよ。」
  <br/>
  「御冗談で。」
  <br/>
  「まったくさ。お爺さんも、そう思うだろう。」
  <br/>
  「手前でございますか。手前なら、そう云う運はまっぴらでございますな。」
  <br/>
  「へええ、そうかね。私なら、二つ返事で、
  <ruby>
   <rb>
    授
   </rb>
   <rp>
    （
   </rp>
   <rt>
    さず
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  けて頂くがね。」
  <br/>
  「じゃ観音様を、御信心なさいまし。」
  <br/>
  「そうそう、
  <ruby>
   <rb>
    明日
   </rb>
   <rp>
    （
   </rp>
   <rt>
    あす
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  から私も、お
  <ruby>
   <rb>
    籠
   </rb>
   <rp>
    （
   </rp>
   <rt>
    こもり
   </rt>
   <rp>
    ）
   </rp>
  </ruby>
  でもしようよ。」
  <br/>
  <div class="chitsuki_1" style="text-align:right; margin-right: 1em">
   （大正五年十二月）
  </div>
